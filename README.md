# Biblioteca BD

Projeto acadêmico desenvolvido para a disciplina de Banco de Dados, utilizando PostgreSQL e Sequelize para gerenciamento de uma biblioteca.

## Objetivo

Desenvolver um banco de dados relacional capaz de controlar:

- Usuários
- Livros
- Categorias
- Empréstimos

## Tecnologias Utilizadas

- Node.js
- Express
- PostgreSQL
- Sequelize

## Estrutura do Projeto

```
biblioteca-bd/
│
├── app/
├── config/
├── justificativa/
├── modelagem/
├── queries/
├── scripts/
├── server.js
└── README.md
```

## Funcionalidades

- Cadastro de usuários
- Cadastro de categorias
- Cadastro de livros
- Controle de empréstimos
- Consulta de livros disponíveis
